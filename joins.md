# SQL JOINS

## DEFINITION
<!-- REF: Devart and Uncodemy -->

<!-- ref: w3 scholl -->

- JOIN clause is used to combine rows from two or more tables, based on a related column between them.

# Inner join
- Returns all records from left table and all matching from the right otherwise NULL

# Outer join
- Returns all records from right table and all matching from the left otherwise NULL

# Full join
- Returns all matching records
